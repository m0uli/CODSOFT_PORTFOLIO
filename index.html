MY PROJECTS

HEALTHIFY

A user friendly mental health care website
come chatroom that works with you and for you.
It is made using HTML, CSS and react.js. Here
users can anonymously share their thoughts and
feelings and also chat with people all over the
world and at the same time can avail counselling
sessions with therapists on the website.

VIRTUAL MOUSE

Made using Python and its packages, through
this project we can operate the mouse virtually
using hand gestures. Operations like selecting,
dragging, double click, etc can be performed
just using our fingers. We can also regulate the
sound by bringing our fingers close (lower the
volume) and far apart (increasing the volume).

E-VOTING

This evoting platform which is made using
Javascript and solidity, uses blockchain to
provide a safe and accurate, at the same time,
faster alternative to the traditional technologies
of voting using paper ballots and EVMs. This
new election system is critical to limit fraud
and having the voting process traceable and
verifiable.
